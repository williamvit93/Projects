ADOTE UM PET

A ONG "ADOTE UM PET" é uma organização não governamental sem fins lucrativos que tem o compromisso de destinar 65% de todos os recursos arrecadados às atividades inerentes ao cuidado dos animais em estado de abandono. Os demais 35% são utilizados para despesas administrativas. O uso dos recursos da "ADOTE UM PET" é rigorosamente controlado.
